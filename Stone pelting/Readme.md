# Stone-pelting-image-classification

Primary goal : To identify the stone pelting action in an image and classify accordingly.

Solution : Data of 50 images were captured to build a CNN model for detection of stone pelting in an image.

Result : Used Inception V3 which performed well with 91% of accuracy score.
